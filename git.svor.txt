#Git spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd - notað til að beyta núverandi directory
	cd ~ - notað til að fara í heima directory
	ls - sýnir lista yfir öllu í directory
	pwd - print working directory
	mkdir - býr til directory
2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone - clónar skrár frá git hub
	git log -  sýnir hvaða breytingar hafa verið gerðar við skrá
	git status - sýnir hvað hefur verið breytt nýlega og á eftir að commita, almenn status
	git diff - sýnir breytingar gerðar við indexið
	git checkout - skiptir um branch
3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff - sýnir almennar breytingar
	b) git diff --staged  - sýnir allar sviðslegar breytingar
	c) git diff commit1 commit2 - sýnir breytingar á milli tvem innsetningum
4. 	Hvað er útgáfustýring (e.version control)? 
	- tekur upp allar breytingar gerðar við skrár, maður getur kallað á akveðna útgáfustýringu.
5.	Hverjir eru helstu kostir við að nota GIT?
	-helstu kosturinn er samvinnan, að geta sókt skrá og prufað að breyta henni án þess að breyta hjá hinum, vista svo breytinguna á masterinn og breyta hjá hinum, einfalt og þæginlegt.
6.  Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
 	-eins oft og hægt er, ef eitthvað klikkar þá er hægt að sækja siðustu breytingu aftur, gott að commita eftir nokkrar breytingar.
 7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í 
 	Git?
 	-Working Directory: mappan sem maður er að vinna með, þar sem skrárnar eru.
 	-Staging area: svæði þar sem breytingar eru geymdar, þú ákveður svo hvort þær séu staðfestar
 8.	Hvenær er sniðugt að nota greinar (branches)? 
 	-þegar maður er að vinna með örðum og þegr maður er með fleyri en eitt verkefni, þá getur maður skilað inn breytingum á annað branch.
